## Getting Started

## Frontend

1- Clone o repositório

2- Vá para a pasta do projeto

```cd dti-project```

3 - Instale as dependências

```npm install```

4 - Rode o frontend

```npm run dev```

## API

1 - Vá para a pasta da API

```cd api```

2 - Instale as depedências

```npm install```

3 - Inicie a API

```npm start```
